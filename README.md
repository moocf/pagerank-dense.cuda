## Usage

```bash
# Download program
rm -rf pagerank-dense
git clone https://github.com/cudaf/pagerank-dense
```

```bash
# Run
cd pagerank-dense && nvcc -Xcompiler -fopenmp -O3 main.cu && nvprof ./a.out data/aug2d.mtx
```
